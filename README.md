# Asunción Investment Tracker

Seguimiento de portafolio de inversiones en la **Bolsa de Valores de Asunción, Paraguay**.

## Características

- 📊 **Dashboard** - Vista general del portafolio con gráficos interactivos
- 💰 **CashFlow** - Proyección de flujos de caja y pagos esperados
- 🔔 **Alerts** - Alertas de vencimientos y eventos importantes
- 📰 **NewsFeed** - Noticias relacionadas con emisoras del portafolio
- 🚀 **BondLaunches** - Lanzamientos de bonos disponibles
- 📈 **InvestmentManager** - Gestión completa de inversiones

## Stack Tecnológico

- **Frontend:** React 19 + TypeScript
- **Build:** Vite 8
- **Desktop:** Electron 36
- **UI:** Lucide React icons
